Building Envelope + Pavilion Design

2D + 3D structural explorations

The building envelope iteration series began with a double-screen pattern. A single square pattern was repeated 16 times to encompass a building fa√ßade. A second design was to be repeated in the same way behind the first design. This way, the design blocks 70% of daylight from entering the building in a creative way.




The second iteration was to create a free-standing pattern with only interlocking joints; no adhesive materials were to be used. 2D laser-cut pieces are the primary material for this pattern. Variations occur between the joints, leading to a very interesting pattern. However, when looked at in plan, it is a basic two-way grid of perpindicular lines.


The third and final iteration is a pavilion designed with the skills exhibited in the previous two iterations. The structure is essentially created via interlocking 2D pieces, but located at the intersections are original joints designed to accomodate the various sizes and shapes of intersections.
